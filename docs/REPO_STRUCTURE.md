# Lithic Ecosystem Monorepo Structure

This repository is a scaffold intended to evolve into the full Lithic ecosystem.
Major modules:

- `packages/compiler/*` — lithc, lithfmt, lithlint, IR toolkit
- `packages/vm/*` — LithoVM runtime, bytecode format, receipts, zk verifier
- `packages/lscl/*` — Secure contracts library (Lithic source modules)
- `packages/provider/*` — Reference AI provider node + SDK placeholders
- `packages/zk/*` — Circuit templates + registry tooling
- `packages/testing/*` — devnet/mocks/fuzz/fixtures
- `apps/docs/*` — docs.lithosphere.network source scaffold
- `apps/examples/*` — example contracts and apps
